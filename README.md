# Haskell Indent

A simple vim plugin concerned only with proper automatic indentation for 
Haskell file types.

## Installation

- Use [Vundle](https://github.com/gmarik/Vundle.vim).
- Add `Plugin "pbrisbin/vim-hs-indent"` to `~/.vimrc`.

## Usage

- Edit a Haskell file.

## (Planned) Features

- Various lists (exports, records, etc)
- `let` expressions
- `if` expressions
- `case` expressions
- `do` syntax
- `where` clauses
